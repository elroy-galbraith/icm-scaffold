# Glossary (Layer 3)

Domain terms used consistently across all stages. Add terms during setup or as they
emerge in research. Stage contracts require: define these terms once, use them exactly.

| Term | Definition | Notes |
|---|---|---|
| AI agent (support context) | Software that autonomously handles a customer support interaction end-to-end — understanding intent, taking actions (e.g., issuing a refund, tracking a shipment), not just suggesting a reply | Distinguish from simple chatbots/scripted bots |
| Agentic AI | AI systems that can plan and execute multi-step tasks with some autonomy, as opposed to single-turn Q&A | Used to distinguish 2025-era support tools from older rule-based chatbots |
| Deflection rate | Share of support contacts resolved without reaching a human agent | Key automation KPI; watch for vendor-inflated figures |
| CSAT | Customer satisfaction score, usually a post-interaction survey rating | Common metric cited alongside automation performance |
| Human-in-the-loop (HITL) | Workflow design where an AI drafts or proposes an action but a human agent approves or can intervene before it completes | Common risk-mitigation pattern for early-stage pilots |
